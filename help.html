Help

Registration

Plumbers certified for testing backflow prevention assemblies who would like to participate in the program need to fill out the "Certified BackFlow Prevention Assembly Tester" form and then it must be kept up to date.

Your Information

Please review your information found on the home tab and contact the Cross Control Connection Division about any descrepencies. If your license, backflow certification, insurance or test kit calibration expires you will not be allowed in the system until your information has been updated in the system using the authorization form listed above!

Passwords

Don't lose your password. You can change your password to whatever you want on the Home tab. No one has access to your password! We can reset your password but can not tell you what it is.

Searching

The application allows users to search both the current backflow assembly inventory and the testing history. A search in the inventory will return information about each individual bfa. A search in the testing history will return test specific information.

Some data for the test history entered prior to the online bfa testing and inventory application may be incomplete, specifically test result psi values and the tester's name.

Inspection

Backflow assembly must be in the Manatee County Backflow inventory in order to be recertified or replaced. If you have discovered a backflow assembly not in our inventory then you must use the new assembly form.

Recertify Form

If the backflow assembly has successfully passed the certification tests with or without repair, then you must register the test results using the recertify form.

Replace Form

If the backflow assembly could not pass its certfication tests, either through repair or otherwise, you can replace the backflow prevention assembly with a new unit and submit the tests for the new information.

New Form

Manatee County wants to know the location of all backflow prevention assemblies. When home owners and contractors do not pull a permit for installing backflow assemblies, the County does not know about these assemblies. Therefore, if a user discovers a backflow prevention assembly not currently in our inventory, please use the new assembly form to report this backflow assembly to the Manatee County Utility Customer Service department.

Report Incorrect BFA Data Form

If you have accidently submitted wrong data or if you have discovered indescrepencies with a current bfa in inventory, use this form to tell us. Tell us whatever you can so that we can fix any problems with the data.

Inspection Report

Use this form to view and print test results captured in the system. The report is available as a printable web page or and adobe pdf. The web page has been formatted for nice looking prints directly from your internet browser. The Adobe Acrobat file allows you to view the report offline, save it for your records or email it to clients.

Feedback

Please contact us if you have questions, problems, concerns or suggestions! We want to make the Backflow Prevention Assembly Testing and Inventory Application something that helps both Manatee County's Citizens through a safer drinking water supply with well maintained data and helping the backflow tester's who's responsibility it is to ensure the maintenance of these assemblies.
